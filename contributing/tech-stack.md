# Technology Stack: rust-yscraper

## Backend
- **Language**: Rust
- **Framework**: Axum (web framework)
- **Runtime**: Tokio (asynchronous runtime)
- **API Documentation**: OpenAPI / Swagger (via `api_gen` and `web_server`)

## Frontend
- **Language**: TypeScript
- **Library**: React
- **Build Tool**: Vite
- **Styling**: Material UI (as per product guidelines)

## Database
- **Engine**: PostgreSQL
- **Migration Tool**: Liquibase

## Infrastructure
- **Containerization**: Docker
- **Orchestration**: Kubernetes (k8s)
- **CI/CD**: GitHub Actions (defined in `.github/workflows`)

## Development Tools
- **Package Manager**: Cargo (Rust), npm (Frontend)
- **Formatting**: `cargo fmt` (Rust), Prettier/ESLint (Frontend)
